Answers to Quention No. 107

Q107. Which HTML Tag generally represents the top section of a webpage and what it usually contains.?

Answer: 

<header> HTML Tag generally represents the top section of a webpage. It contains
  • Website Title
  • Logo
  • Introductory Content

Q108. Which HTML Tag Represents navigation links?

Answer:

<nav> HTML Tag represents navigation Links

Q109. Which HTML Tag Represents the primary content?

Answers:

<main>
HTML Tag represents the primary content. A website usually has one main content.

Q110. Which HTML Tag Groups related content?

Answers:

<section>
HTML Tag groups related content together.

Q111. Which HTML Tag Represents a self-contained piece of content?

Answers:

<article>
HTML Tag represents a self-contained piece of content.

Q112. Which HTML Tag Represents the bottom section of a webpage?

Answer:

<footer>
HTML Tag represents the bottom section of a webpage.
