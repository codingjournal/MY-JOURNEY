Create basic table structure and explain each tag.

Answer

            
<table>
    <tr>
        <th>Subject</th>
    </tr>
    <tr>
        <td>HTML</td>
    </tr>
</table>

<table> - It creates a Table.
<tr> - It creates Row of a Table.
<th> - It creates Header of the Table.
<td> - Each <td> represnts a Column in the Table Structure and the text in between the Tag represents data of the Table.
