Use of Head Tag

In HTML, there are six Head Tag <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.
We can use these Head Tags to format our Writeup and put Section Headings. We should always use Heading Tag in order from <h1> to <h6>
