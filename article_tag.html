How to use <article> tag and why to use it?

Article Tag in HTML is a semantic element. It tells the browser that the content inside the article tag is a article type in nature. It is just not a generic box rather it is a meaningful box.

It makes your code cleaner to read. You can easily identify where a piece of content starts and end. Screen readers which are used by Visually Impaired users to browse the web can find the main content easily in the webpages. It also helps in Search Engine Optimization.

Content inside of an article tag should be such that if you removed it from your website and put it in another page, it should still make perfect sense to a reader. Content inside it should be independent and reusable.
